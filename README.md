# huddle 🎓
### Built by students, for students.

**huddle** is a unified digital infrastructure designed to eliminate the 'hidden costs' of student life. 

## 🚀 The Vision
In a world of expensive subscriptions and fragmented school portals, huddle provides:
* **Crowdsourced Intelligence:** Real-time library occupancy and noise levels.
* **Academic Sync:** Shared syllabus tracking and deadline management.
* **Peer Economy:** A verified marketplace for textbook relays and equipment sharing.

## 🛠️ Tech Stack (In Progress)
* **Frontend:** Next.js / React
* **Backend:** Supabase (PostgreSQL)
* **Authentication:** GitHub/School Email OAuth
* **Hosting:** Vercel
